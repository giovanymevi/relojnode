# Reloj Digital con Node.js

Este es un proyecto sencillo de un reloj digital en tiempo real. Utiliza **Node.js** y **Express** para el servidor backend, sirviendo una interfaz frontend construida con **HTML**, **CSS** y **JavaScript**.

## 🛠 Requisitos Previos

Para ejecutar este proyecto, necesitas tener instalado en tu computadora:

*   [Node.js](https://nodejs.org/) (versión 12 o superior recomendada)
*   npm (normalmente viene instalado con Node.js)

## 🚀 Instalación

1.  **Clona o descarga** este repositorio en tu máquina local.
2.  Abre una terminal en la carpeta raíz del proyecto.
3.  Ejecuta el siguiente comando para instalar las dependencias necesarias (Express):

```bash
npm install
